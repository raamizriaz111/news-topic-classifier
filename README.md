# News Topic Classifier using BERT

## Objective
Build a machine learning model that classifies news headlines into 4 categories using a fine tuned BERT model.

---

## Dataset
AG News Dataset (from Hugging Face Datasets)

Classes:
- World
- Sports
- Business
- Sci Tech

---

## Methodology

1. Loaded AG News dataset
2. Tokenized text using BERT tokenizer
3. Fine tuned bert-base-uncased model
4. Trained using Hugging Face Trainer API
5. Evaluated using Accuracy and F1 Score
6. Built a live web app using Gradio

---

## Tools & Technologies
- Python
- BERT (Transformers)
- Hugging Face Transformers
- Hugging Face Datasets
- Gradio
- Google Colab
